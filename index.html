Welcome to OpenShift with NodeFly

Signing up for NodeFly is simple

Just go through the sign up process through the NodeFly website.

Install nodefly via npm with npm install -S nodefly.

Then add/update the following code in your webserver:

var app_name = process.env.OPENSHIFT_APP_NAME || 'local_development',
    host_url = process.env.OPENSHIFT_APP_DNS  || 'localhost',
    gear_id = process.env.OPENSHIFT_GEAR_UUID || 1,
    options = {};

require('nodefly').profile(
  '00000000000000000000000000000000000000000', // <-- enter your nodefly dev key
  [ app_name,                                  // See http://nodefly.com/#howto 
    host_url,
    gear_id], // to identify multiple gears or processes (for scaled apps)
  options // optional
);

Next, commit your changes and update your OpenShift app:

git add package.json server.js
git commit -m 'setting my nodefly developer key, adding realtime app monitoring'
git push

You should now be able to monitor your application performance at http://nodefly.com/#dashboard

Need help? Reach out to support@nodefly.com and we would be happy to help!
